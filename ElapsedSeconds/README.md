## Function(s) Usage:
	elapsedSeconds(startDate, endDate);

## Inputs:
	Two times or dates, formatted to JavaScript, Python, Ruby, C# or Nim

## Returns:
	An integer representing the number of seconds between the times or dates

## Solution Complexity:
	Time: O(1) - Constant
	Space: O(1) - Constant
